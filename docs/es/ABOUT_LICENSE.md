## Qué significa la Licencia MIT

La Licencia MIT es una de las licencias de código abierto más permisivas y utilizadas. Esto es lo que significa en lenguaje sencillo:

### PUEDES

-  **Usar** el software para cualquier propósito: personal, comercial, académico o de otro tipo.
-  **Copiar** y distribuir el software libremente.
-  **Modificar** el código fuente como desees.
-  **Incorporarlo** en otros proyectos.
-  **Publicarlo**, sublicenciarlo o venderlo.
-  **Usarlo de forma privada** sin compartir tus cambios.

### DEBES

-  **Conservar el aviso de derechos de autor**: cualquier copia o parte sustancial del software debe incluir el aviso de derechos de autor original y el texto de la licencia.
-  **Incluir la licencia**: el archivo «LICENCIA» (o su texto) debe distribuirse con el software.

### NO PUEDES

-  **Responsabilizar al autor**: el software se proporciona "tal cual", sin garantía. El autor no se hace responsable de ningún daño o problema derivado de su uso.
-  **Reclamar la autoría** de la obra original.

## ¿Por qué el MIT?

Se eligió MIT porque CodeShamer está desarrollado por desarrolladores para desarrolladores. El objetivo es alcanzar el máximo alcance: cualquiera debería poder usar, integrar o desarrollar esta herramienta sin problemas legales. Una licencia permisiva fomenta las contribuciones, integraciones y la adopción en todo tipo de proyectos.
